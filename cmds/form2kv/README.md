# form2kv

convert form data to key-value data

## Installation

form2kv is a standalone awk script file with no dependencies, so you can use it by copying it to a suitable location.

## Requirements

awk(gawk/mawk/nawk) and a POSIX-compatible shell.

## Usage

```sh
form2kv <file>
```
