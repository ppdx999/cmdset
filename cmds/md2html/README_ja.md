# md2html

MarkdownをHTMLに変換するコマンド

## Installation

それぞれの言語の実装によってインストール方法が異なります。

例えばawkの場合は、依存のない独立したスクリプトなのでコピーして適当な場所において使ってください。

## Features

[このマークダウン](test.md)を[このHTML](test.html)に変換できます。

## Usage

```sh
md2html test.md > test.html
```
